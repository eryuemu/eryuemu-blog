## 三、注册商选择

### Spaceship（首年首选）

- 优势：首年促销价极低，COMPROS 优惠码可使 .com 首年 $2.90
- 劣势：次年续费按正常价（$10.18），不算最低
- 限制：COMPROS 每个账号限用一次

### Cloudflare（长期持有终点站）

- 优势：**零利润批发价**，不赚差价。.cc 续费 $8.00，.com 续费 $10.46
- 额外要求：必须使用 Cloudflare DNS（免费，反而更快）
- 注意：Cloudflare 不提供首年促销，直接按批发价

### 阿里云（本次不买，保留新人资格）

- `.cn` 后缀：38 元首年 / 42 元续年，国内最便宜
- **致命陷阱**：在阿里云买任何东西（包括域名）都会消费新人资格，导致 38/68 元新人服务器再也买不到
- 策略：**保留阿里云白号**，等真正需要国内服务器时再用

## 四、域名转移策略（薅羊毛路线图）

```
第一年：Spaceship 用优惠码低价买入
    ↓
第 10-11 个月：搜索各平台转入优惠（Transfer Promo）
    ↓
    ├── 有优惠 → 转入优惠平台（自动续费一年）
    └── 无优惠 → 转入 Cloudflare（批发价续费）
```

**注意事项**：
- 新注册域名有 **60 天锁定期**，期间禁止转移
- 转移时支付的一年续费会**叠加**到现有有效期上，不亏时间
- 转移流程很简单：解锁域名 → 获取 Auth Code → 在新平台提交 → 5 天内自动完成
- **绝对不要等域名过期再重新买**：过期后有 70-80 天保护期，期间会被抢注机器人秒杀

## 五、国内域名政策辨析

> Claude 纠正了 Gemini 的一个错误：Gemini 说"阿里云会扫描并阻断未备案的域名"，实际上这套阻断机制只针对**解析到阿里云境内服务器**的情况。

### 真实规则

| 情况 | 是否需要实名 | 是否需要 ICP 备案 |
|------|:--:|:--:|
| 阿里云买 .cn/.com，解析到 **国内服务器** | ✅ 必须 | ✅ 必须 |
| 阿里云买 .cn/.com，解析到 **境外服务器**（Vercel/GitHub Pages） | ✅ 必须 | ❌ 不需要 |
| Spaceship 买 .com/.cc，解析到境外服务器 | ❌ 不需要 | ❌ 不需要 |

- **实名认证**：在国内注册商买域名必须实名（.cn 全球通用这个规定），审核 1-3 个工作日
- **ICP 备案**：仅当域名解析到国内服务器时才触发，境外服务器不走这套
- **公安联网备案**：理论上指向境外服务器也需要，但实际不审核不卡访问，个人博客没人管
- **Spaceship 不卖 .cn**：因为 .cn 由 CNNIC 管理，要求所有注册人实名，海外注册商干脆不做这个后缀

## 六、价格心理建设

- .com 续费每年约 74 元，按 ICANN 规定**每年涨幅上限 7%**，不会暴涨
- 普通域名（如 `eryuemu.com`）一旦注册就是普通价，不会被中途改为溢价域名
- 三年总花费约 169 元，和 .cc 的差价只有 30 元
- Spaceship 搜索结果中的百分比（如 +17.75%）是行业价格偏离度，不是涨价预警
